# STFX — the modular, multi-language foundation for Trust Spanning Protocol & ToIP

**Rust • TypeScript • Python • Kotlin • WASM**

The first complete, high-performance, cleanly-modular reference implementation of ToIP Layers 1–3 and the Trust Spanning Protocol (TSP).

Available today as `stfx-crypto`, `stfx-vid`, `stfx-keys`, `stfx-tsp`, … on [crates.io](https://crates.io)

→ npm `@stfx/*`, PyPI `stfx-tsp`, Maven `org.stfx:*` coming Q1 2026
